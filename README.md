# Email Spam Classifier

A Python-based Email Spam Classifier that detects 
spam messages using Natural Language Processing 
and Machine Learning techniques.

##  Features
- Classifies emails as Spam or Not Spam
- NLP-based text preprocessing pipeline
- TF-IDF Vectorization for feature extraction
- Trained ML model for accurate classification
- Simple and interactive web app interface

##  Technologies Used
- Python
- NLTK
- Scikit-learn
- TF-IDF Vectorization
- Jupyter Notebook
- Streamlit (Web App)

##  How to Run
1. Clone this repository
2. Install requirements: `pip install -r requirements.txt`
3. Run `python app.py`
4. Open browser and test spam detection
